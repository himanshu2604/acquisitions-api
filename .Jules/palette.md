## 2026-04-23 - [Accessibility: Skip-to-content and Focus-visible]
**Learning:** For a landing page with complex interactive elements, providing a "Skip to content" link and explicit `focus-visible` styles dramatically improves the keyboard navigation experience, especially when using utility-first CSS like Tailwind which may have subtle default focus styles.
**Action:** Always include a skip-to-content link and ensure interactive elements have high-contrast focus rings in future projects.
